### Mysql

| 配置         |   描述                                         |      示例         |

| 表名 | 选择表名                                  |           |
| 最大行 | 流程设计阶段预览数据的行数                                  |            |
| 增量读取 | 流程真实运行时根据增量的配置数字类型、id、日期进行增量读取                                  |            |
| 自定义查询 | 输入自定义SQL查询数据,当此项不为空时`表名`将失效                                  |            |
