### 概述
<span>在{docConfig.projectName}使用中为了更方便快捷的管理输入、输出的文件,
    可以添加常用的文件系统。当前支持的协议本地文件(即系统运行所在的系统目录)、SFTP、FTP、SMB、HDFS。
    支持本地批量上传、远程目录打包下载、文件系统间的文件传输、文件预览。
</span>


### 文件系统配置
| 配置  | 描述                              |       示例       |
|-----|:--------------------------------|:--------------:|
| 名称  | 文件系统的名称                         |    这是一个文件系统    |
| 协议  | 为当前新建文件系统选择传输协                  |                |
| 主机  | 除开`本地文件`其余协议的文件系统必须填写文件系统所在主机IP |  192.168.0.1   |
| 端口  | 输入文件系统所在主机端口                    |       22       |
| 用户名 | 输入文件系统认证的用户名                    |                |
| 密码  | 输入文件系统认证的密码                     |                |
| 路径  | 输入默认进入文件系统的路径,必须输入**绝对路径**      | /或/usr或D:/data |
### 创建文件系统
点击**[文件系统>右键空白区域>添加文件系统]**可创建新的文件系统。
![create.jpg](/img/filesystem/create.jpg)

### 更新文件系统

点击**[右键文件系统>更新文件系统]**。
![update.jpg](/img/filesystem/update.jpg)

### 新建目录

双击进入文件系统**[右键空白区域>新建文件夹]**。

### 上传文件

双击进入文件系统**[右键空白区域>上传文件]**。
![upload.jpg](/img/filesystem/upload.jpg)

### 下载与批量下载

双击进入文件系统**[右键文件、目录或者使用左键框选>下载或压缩并下载]**。

### 拷贝与移动

双击进入文件系统**[右键文件、目录或者使用左键框选>拷贝、移动>选择目录>点击确定]**。

双击进入文件系统**[右键文件、目录或者使用左键框选>直接拖拽到目录]**。

### 查看

双击文件或者**[右键文件>查看]**。   支持**csv、pdf、xlsx、txt、img、Js、视频、音频**等文件的查看。
